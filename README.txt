1. 프로그램 구성

	1) 로그인 화면 : 프로그램 접속을 위한 로그인 화면

	2) 메인화면 : 프로그램의 여러가지 기능을 이용할 수 있는 화면

	3) 레지스터 화면 : 유저를 등록하는 화면

	4)  프로젝트 화면 : 프로젝트를 생성하고 볼 수 있는 화면

	5) 이슈 생성 화면 : 이슈를 생성하는 화면

	6) 이슈 상세정보 화면 : 프로젝트를 선택할 시 메인프레임에 이슈들이 등장, 클릭시 상세정보 화면

	7) 검색 화면 : 프로젝트, assignee, state, reporter 별로 검색을 할 수 있는 화면

	8) 트렌드 화면 : 프로젝트, 월별 이슈, 일별 이슈를 보여주는 통계 화면



2. 프로그램 제작자

	BE team : 강시운, 윤민석
	FE team : 김지찬, 유호인, 정용희

★설명이 필요한 프로그램 요소들★

┏━━━━━━━┓
│ 프로젝트 버튼 │
┗━━━━━━━┛
프로젝트 	생성 : 프로젝트의 이름 작성, 멤버 추가, 프로젝트 설명 작성을 완료한 후에 생성버튼을 누릅니다.

프로젝트 보기 : 프로젝트 리스트를 선택하여 불러오면 메인 프레임에 이슈 리스트들이 생성됩니다.
		이슈 리스트중에 하나를 클릭하면 이슈 상세정보를 수정할 수 있는 창이 생성됩니다.
			admin : 모든 권한을 갖고 있습니다.
			Manager : issue state, issue asignee 를 변경할 수 있습니다.
			Tester : issue 생성, state reopened, resolved로 변경할 수 있습니다.
			dev : 자신에게 assign된 issue를 fixed로 변경할 수 있습니다.


┏━━━━━━━┓
│  디테일 버튼   │
┗━━━━━━━┛
메인 프레임에 보여지는 각각의 이슈들에 달린 버튼으로, 클릭시 이슈에 달린 코멘트에 대한
정보를 볼 수 있고 댓글을 추가, 삭제할 수 있습니다.



┏━━━━━━━┓
│ 이슈 상세정보 │
┗━━━━━━━┛
이슈를 클릭하면 나오는 창으로 Issue name, Reporter name, issue assignee name, Fixer name, date
issue type, state 정보가 있고 하단에는 이슈 설명과 댓글 추가 및 삭제가 됩니다.
